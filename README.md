# 台灣扣繳稅額計算器

> 各類所得扣繳稅額精準計算工具（114年度適用）

## ✨ 功能特色

- ✅ **完整規則** - 涵蓋 9 大類所得扣繳規則
- ✅ **精準計算** - Decimal.js 零誤差運算
- ✅ **薪資稅額表** - 完整 114 年度扣繳稅額表
- ✅ **3 步驟操作** - 身份 → 類別 → 計算
- ✅ **PDF 匯出** - 一鍵產生扣繳計算單
- ✅ **離線可用** - 純前端運行，無需伺服器

## 🎯 支援所得類別

### 本國居住者
- 💵 薪資所得（5% / 稅額表）
- 🏢 租金給付（10%）
- 🤝 佣金收入（10%）
- ©️ 權利金收入（10%）
- 🏆 獎金給與（10%）
- 👔 執行業務收入（10%）
- 👋 退職所得（6%，扣除免稅額）
- 📊 股利與盈餘（免扣但須申報）
- 💰 利息所得（10%，可用儲蓄免扣繳憑證）

### 非居住者
- 💵 薪資所得（6% / 18% 分級）
- 🏢 租金給付（20%）
- 🤝 佣金收入（20%）
- ©️ 權利金收入（20%）
- 🏆 獎金給與（20%）
- 👔 執行業務收入（20%）
- 👋 退職所得（18%）
- 📊 股利與盈餘（21%）
- 💰 利息所得（20%）

## 🚀 使用方式

### 線上使用
直接開啟 `index.html` 即可使用（支援 file:// 協定）

### 本地部署
```bash
git clone https://github.com/maotai11/tw-tax-withholding-calc.git
cd tw-tax-withholding-calc
# 雙擊 index.html 或使用瀏覽器開啟
```

## 📖 法規依據

- 《所得稅法》第 88 條
- 《各類所得扣繳率標準》（114 年度）
- 財政部國稅局扣繳申報規定
